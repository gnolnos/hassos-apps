# hassos-apps

Collection of Home Assistant add-ons.

## Wyoming Vietnamese ASR

Vietnamese speech recognition using Wyoming protocol.

- Add-on version: 2026.3.1
- Model: Zipformer-30M-RNNT-6000h (WER 7.97%)
- Repository: https://github.com/gnolnos/hassos-apps

### Install

1. Add repository: `https://github.com/gnolnos/hassos-apps` in Add-on Store
2. Install "Wyoming Vietnamese ASR"
3. Start and configure Wyoming integration in HA to point to `homeassistant:10400`.
